## 👀 프로젝트 소개
<a name="프로젝트-소개"></a>
- 주제: SHOONG은 좋아하는 그룹의 포토카드를 모으고 관리할 수 있는 콜렉트북과 아이돌 생일 카페 정보를 제공하여 건전한 팬 문화를 지향하는 플랫폼입니다.
- 진행 기간: 2024/02/17 - 2024/03/14 
- 리팩토링: 2024/03/19 - 2024/04/30
